文字提示 Tooltip #

常用于展示鼠标 反馈 时的提示信息

基本使用 #

Details
html
<template>
  <ik-tooltip placement="top">
    <template #content>
      <div>这是提示内容</div>
    </template>
    <ik-button>按钮</ik-button>
  </ik-tooltip>
</template>

基础配置 #

不显示箭头，点击触发展示。可配置白色背景。

点击展开
html
<template>
  <ik-tooltip placement="top" triggerType="click" :arrow="false">
    <template #content>
      <div>这是提示内容</div>
    </template>
    <ik-button>按钮</ik-button>
  </ik-tooltip>
</template>
